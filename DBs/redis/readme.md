# redis

## Install

### macOS

> make sure to have homebrew installed & updated

```
$ brew install redis
$ brew uninstall redis
```

## Commands

> enter redis cli: `redis-cli`

- set key: `SET name reed` / `SET age 1000000`
- delete key: `DEL name` / `DEL age`

- get all keys: `KEYS *`
- flush all keys: `FLUSHALL`
